## Digital Portfolio

A modern, responsive digital portfolio website built with HTML, CSS, and JavaScript. Features a clean design, smooth animations, and full mobile responsiveness.

# Feature

1. Responsive Design - Works perfectly on all devices (desktop, tablet, mobile)

2. Modern UI/UX - Clean and professional design with smooth animations

3. Smooth Scrolling - Seamless navigation between sections

4. Interactive Elements - Hover effects and animated sections
 
5. Contact Form - Functional contact form with validation

6. Mobile-Friendly - Hamburger menu for mobile devices
